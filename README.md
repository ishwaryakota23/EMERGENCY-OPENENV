# Emergency Response Simulation Environment

## Overview
This project simulates a real-world emergency response system where an AI agent must classify emergencies and allocate resources efficiently.

## Features
- Multi-step decision-making (type → severity → resource)
- Reward shaping with penalties
- Dynamic resource allocation based on distance
- Handles extreme scenarios with limited resources

## Tasks
1. Easy: Basic classification
2. Medium: Fire scenario
3. Hard: Accident handling
4. Extreme: Limited resources scenario

## How to Run
```bash
python inference.py
